Página de Teste - Ferramenta de Mover

Esta página serve para testar a funcionalidade da ferramenta de mover elementos na extensão NZR DevTool.

Para testar:

  1. Clique no botão "Anotar e Capturar Tela" da extensão
  2. Use qualquer ferramenta para criar elementos (texto, seleções, formas)
  3. Selecione a ferramenta de mover (✋) ou pressione M
  4. Tente arrastar os elementos criados

Os elementos devem ser movíveis quando a ferramenta de mover estiver ativa.
