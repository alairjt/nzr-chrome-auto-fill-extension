Política de Privacidade

Transparência e proteção dos seus dados

📅 Última atualização: 16 de agosto de 2025
🛡️

1. Introdução

A extensão NZR DevTool foi criada para ajudar no preenchimento automático de formulários e anotação de páginas web. Esta Política de Privacidade descreve como tratamos informações ao usar a extensão, priorizando sempre a sua privacidade e segurança.

📊

2. Coleta e processamento de dados

A extensão executa scripts de conteúdo na página para ler a estrutura do formulário (rótulos, placeholders, tipos de campos e, quando necessário, trechos de contexto visível). Esse processamento ocorre localmente no seu navegador.

Quando você aciona o preenchimento com IA, a extensão monta um prompt contendo apenas metadados necessários dos campos (por exemplo: label, placeholder, tipo e opções do select) e um resumo de contexto textual do entorno visível do campo para orientar a IA. O conteúdo enviado é o mínimo necessário para gerar sugestões relevantes.

🤖

3. Provedores de IA

A extensão pode usar provedores de IA escolhidos por você nas Opções, como OpenAI ou Google Gemini. As chamadas são feitas diretamente do seu navegador para as APIs dos provedores. Suas chaves de API são armazenadas no chrome.storage.sync do próprio navegador e não são transmitidas ao desenvolvedor da extensão.

  • OpenAI API: https://api.openai.com/
  • Google Generative Language API (Gemini): https://generativelanguage.googleapis.com/
    • 💾

    4. Armazenamento

    • Configurações (provedor, modelo, chaves de API): guardadas em chrome.storage.sync.
    • Preferências de interface: tipos de dados visíveis no painel, idioma.
    • Nenhum dado sensível de formulários é salvo pela extensão.
    • Anotações de tela são processadas localmente e não armazenadas permanentemente.
    🤝

    5. Compartilhamento

    A extensão não vende dados dos usuários e não compartilha dados com terceiros além dos provedores de IA selecionados por você, quando o preenchimento com IA é acionado. Todos os dados permanecem sob seu controle.

    🍪

    6. Cookies e rastreamento

    A extensão não cria cookies próprios nem realiza rastreamento de usuários. No entanto, as páginas que você visita e os provedores de IA podem usar cookies e rastreadores segundo suas próprias políticas de privacidade.

    🔒

    7. Segurança

    Adotamos o princípio de minimização de dados no prompt enviado aos provedores de IA e utilizamos as APIs oficiais (HTTPS). Recomendamos manter seu navegador e a extensão sempre atualizados para garantir a máxima segurança.

    ⚙️

    8. Seus controles

    Habilitar/Desabilitar IA Controle completo sobre quando usar os recursos de IA nas Opções
    Gerenciar Chaves de API Edite ou remova suas chaves de API a qualquer momento
    Controlar Envios Evite envios para IA simplesmente não acionando o preenchimento inteligente
    Proteção de Dados Campos preenchidos não são sobrescritos automaticamente
    Configurar Dados Visíveis Escolha quais tipos de dados aparecem no painel
    Modo Anotação Use ferramentas de anotação sem envio de dados externos
    📧

    9. Contato

    Para dúvidas, sugestões ou solicitações relacionadas à privacidade, você pode entrar em contato através dos canais oficiais da extensão. Valorizamos o feedback dos usuários e estamos comprometidos em manter a transparência sobre nossas práticas de privacidade.

    🔄

    10. Alterações nesta política

    Podemos atualizar esta Política de Privacidade periodicamente para refletir mudanças em nossas práticas ou por outros motivos operacionais, legais ou regulamentares. Alterações relevantes serão publicadas nesta página com data de atualização. Recomendamos revisar esta política regularmente.
